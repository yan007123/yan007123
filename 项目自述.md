## 一句话简介

> 和离那日，太子妃沈雾意外听到了废太子萧渊的心声——那个对她冷言冷语的男人，心里想的却是"不要签，求你别离开我"。

---

## 核心卖点

1. **心声反差萌**：表面冷漠的男主 vs 内心疯狂打脸的真心
2. **双向奔赴**：一个想追妻火葬场，一个想验证真假，最终双双沦陷
3. **权谋暗线**：废太子流放路上暗中布局，复位之路步步惊心
4. **甜虐交织**：日常糖多，但有刀子（沈雾的童年创伤、萧渊的母仇）

## 核心人物（5个）

### 沈雾（女主）
- 身份：礼部尚书嫡女 → 太子妃（已和离）→ 流放随行人 → 未来皇后
- 年龄：19岁
- 核心特质：清冷隐忍，内心清醒，幼年丧母创伤，苗疆血统
- 关键能力：医术、观察力、同心蛊读心
- 当前状态：跟随萧渊流放，验证心声真假中

### 萧渊（男主）
- 身份：太子 → 废太子 → 流放犯 → 未来皇帝
- 年龄：24岁
- 核心特质：极致隐忍，算计深沉，被背叛过不敢爱人
- 关键背景：母后被毒杀，兄弟被暗杀，柳氏背叛
- 当前状态：流放黔州，暗中筹划反击

### 青黛（女配）
- 身份：沈雾贴身丫鬟
- 核心特质：活泼忠心，后期与影七有感情线
- 功能：吐槽担当，推进感情线

### 萧承（反派）
- 身份：三皇子，监国，实际掌权者
- 核心特质：心狠手辣，斩草除根必尽
- 当前状态：京城掌权，追杀萧渊

### 赵皇后（反派后台）
- 身份：皇后，萧承支持者
- 核心特质：老谋深算，萧承的幕后军师

## 核心设定：同心蛊

### 运转规则
- 萧渊情绪波动时，沈雾能听到他的心声
- 单向传递：沈雾能听，萧渊不知道
- 距离限制：10米内最强，100米中等，1000米微弱，超出无效

### 副作用（重要伏笔）
1. 偶发性头痛（初期）
2. 耳鸣、失眠（中期）
3. 记忆混淆（后期）
4. 短暂失意识（临界）

### 解蛊方式
- 蛊源死亡（萧渊死）
- 真心血解蛊（需苗疆圣女主持）

## 创作风格指引

### 频道：女频 / 古代言情

### 叙事核心
- 心声描写是灵魂：每章至少3处心声与言语的反差
- 五感俱全：尤其强化嗅觉和触觉
- 推拉美学：甜一下必须虐一下，虐完必须更甜
- 情感具象化：写生理反应，不写"她很伤心"

### 去AI味要点
- 句式必须变化：长短交替，禁止连续3句相同结构
- 词汇离散度：800字内同一概念用3种表达
- 节奏波动：长段→短段→超短段，不规则
- 语义毛刺：每千字0-2处矛盾修饰/通感错位
- 微瑕疵：每2000字注入3处轻微重复/视角漂移

### 禁止清单
- ✗ 直接命名情感（"她很悲伤"）
- ✗ 陈词滥调比喻（"心如刀绞"）
- ✗ AI高频词（"值得注意的是""此外""总而言之"）
- ✗ 固定开头模式（每5章必须换一种开头方式）

## 写作时的重要提醒

### 必须记住
1. 沈雾有苗疆血统（苗疆旁系后人），这是同心蛊能生效的原因
2. 萧渊母后死于毒杀，与二皇子萧衍之死有关
3. 萧渊三年不碰沈雾，是因为不敢信任（柳氏背叛创伤）
4. 沈雾幼年丧母是被继母慢待而死，这是她的核心创伤

### 需要逐步揭示
1. 沈家与萧承的勾连（隐藏在日常中）
2. 影卫组织的存在和势力（中期重要）
3. 谢家与萧渊的联络（复位暗线）
4. 四皇子萧煜的真实身份（40章级别悬念）
