## 一、对话的本质：潜台词大于说出来的话

### 核心原则

女频对话的核心是**潜台词**。
读者读对话，不是读字面意思，而是读字面背后的"没说的话"。

**坏对话**：说什么是什么
```
"我喜欢你。"他说。
"我也喜欢你。"她回答。
```

**好对话**：说A，指B，读者懂C
```
"你今天怎么来了？"她问。
他没答，只是把伞收了，靠在门框上。
"外面雨大。"
她看了眼窗外——明明是晴天。
```

## 二、对话三层次模型

### 第一层：字面意思（说什么）

最简单的信息传递。
```
"你的茶凉了。"  →  茶温度低
```

### 第二层：情感态度（怎么说的）

同样的字面，不同的语气，不同的意思。
```
"你的茶凉了。"（关心）  →  想给你换一杯
"你的茶凉了。"（嫌弃）  →  你怎么连喝茶都不会
"你的茶凉了。"（试探）  →  想看你什么反应
```

### 第三层：潜台词（没说什么）

对话背后藏着真正想说的话。
```
"你的茶凉了。"
"嗯。"
"要我换一杯吗？"
"不用。"
"那我去给萧公子送茶了。"
她站起来，走过他身边时，他突然开口：
"……你今天非要去吗？"
```

**分析**：
- 字面：问茶
- 态度：犹豫、不舍
- 潜台词：你能不能不去见他

## 三、心声与言语的反差（女频核心技法）

### 原理

女频的特殊性在于"心声系统"——读者能看到角色心里想的。
对话与心声的反差，是女频最核心的甜/虐来源。

### 甜蜜型反差

```
心里想的是A，说出来的是B，读者感受到的是甜

【例子】
沈雾：心里"他好温柔"，嘴上"……还行吧"。
读者：磕到了。
```

```
沈雾垂着眼，没看他。
"……谢谢。"
心里想的是：他的手好暖。他身上有雪松的味道。他刚才是不是在看我。
嘴上说的是："路不太滑，不用扶。"
```

### 虐心型反差

```
心里想的是A，说出来的是B，读者感受到的是虐

【例子】
萧渊：心里"我喜欢你"，嘴上"……随你"。
读者：意难平。
```

```
"你不必跟来。"
萧渊的声音很淡，像这山间的风。
心里想的是：这条路太苦了，她跟一天就要受一天的罪。
嘴上说的是："我不会照顾人。"
沈雾听到的心声是：但是你可以留下。
```

## 四、对话设计检查清单

### 每句对话都要过三关

| 关卡 | 问题 | 通过条件 |
|-----|------|---------|
| 第一关 | 这句话有没有字面以外的意思？ | 没有→删/改 |
| 第二关 | 这句话体现了说话人什么性格/当前什么情绪？ | 体现不了→改 |
| 第三关 | 这句话有没有给下一句对话留钩子？ | 没留→改 |

### 反差设计（心声+言语）

```
【甜蜜型】
□ 心里想的和嘴上说的是反的
□ 读者能读出潜台词
□ 有1-2层解读空间

【虐心型】
□ 明明想靠近，说出来的是推开
□ 明明在意，表现出来的是冷淡
□ 读者知道但角色不知道（信息差）
```

### 潜台词设计

```
□ 这句话有"没说的话"支撑
□ "没说的话"比"说的话"更重要
□ 对话结束后，读者能回味
```

## 五、对话节奏：短-短-长的波浪

### 核心原则

对话不能一长串，也不能全是短句。
要像心跳一样，有快有慢，有疏有密。

### 公式

```
短句（紧张/冲突）→ 短句（压制）→ 长句（释放/转折）
```

### 示例

```
【短-短-长节奏】

"你叫什么名字？"
"……沈雾。"
"哪个雾？"
"云雾的雾。"
他没再问，只是看着她。她被他看得有些不自在，垂下眼——
"黔州冷，你身子骨弱。"
他说得很随意，像在聊天气。
```

**分析**：
- "你叫什么名字？" → 短，简洁
- "……沈雾。" → 更短，犹豫
- "哪个雾？" → 短，追问
- "云雾的雾。" → 短句
- （叙述描写，约30字）
- "黔州冷，你身子骨弱。" → 长，关心

## 六、对话中的沉默

### 核心原则

有时候沉默比说话更有力。
对话中的"留白"，是读者想象力入场的时候。

### 类型

#### 1. 问答之间的沉默

```
"你为什么要帮我？"
她等着他回答。
他没有说话。
风吹过，帐篷的帘子晃了晃。
"……因为欠沈家的。"他终于开口。
```

**分析**：沉默让答案更沉重

#### 2. 说完话之后的留白

```
"这条路很危险。"
"我知道。"
"你会后悔的。"
"……"
她没回答，只是笑了笑。
```

**分析**：沉默传递"我不会后悔"的坚定

#### 3. 打断式沉默

```
"你听我解释——"
"不用了。"
两句话，中间隔了整整三秒的沉默。
```

**分析**：沉默比任何解释都更有力

## 七、对话与人物性格

### 核心原则

对话是人物性格的外化。
同一句话，不同性格的人说出来完全不同。

### 性格对照表

| 性格 | 说话特点 | 举例 |
|-----|---------|------|
| 外冷内热 | 话少、短、冷，但关键时刻说出的话很有分量 | "随你。" |
| 活泼开朗 | 话多、语气词多、爱用感叹句 | "真的吗！太棒了！" |
| 城府深 | 话不说完、喜欢反问、爱绕弯子 | "你觉得呢？" |
| 直率 | 有话直说、不绕弯子、偶尔伤人 | "我不喜欢你这件衣服。" |
| 傲娇 | 嘴硬心软、正话反说 | "谁要你管啊。" |

### 同一场景不同性格

**场景**：对方给了她一块糕点

```
【外冷内热型萧渊】
"吃点东西。"
沈雾看了眼那块糕点——是她喜欢的桂花糕。
她刚想说谢谢，就听见他补了一句：
"……别多想，路上的干粮而已。"
心里想的是：记得她爱吃甜的。

【直率型配角】
"给你！路上垫垫肚子！"
"谢谢。"
"不客气！你喜欢吃甜的吧，我看你之前盯着那家糕点铺看了好久！"
```

## 八、女频对话的特殊性

### 古代言情的对话特点

```
1. 称谓讲究
   - 对陌生人用"您"
   - 对长辈用敬称
   - 对平辈/晚辈可以随意
   - 生气时可能直接叫名字

2. 说话绕弯子
   - 不直接说"我喜欢你"
   - 用"你今天真好看"代替
   - 用"你粥凉了"暗示关心

3. 身份感强
   - 主仆对话有明确分野
   - 男女授受不亲的边界感
   - 礼仪规矩在对话中的体现
```

### 心声配合技巧

```
女频特有的"三明治结构"：

第一层（表）：言语/行动
第二层（里）：心声
第三层（表）：言语/行动的反转

【例子】
"你的脸怎么红了？"
沈雾伸手探了探自己的脸颊——烫的。
她偏过头，声音发紧："……车里闷。"

心里想的是：他在看我。他什么时候在看我的。
```

## 九、对话中的"钩子"

### 核心原则

每段对话结尾要留一个钩子，让读者想看下一句。

### 钩子类型

| 类型 | 说明 | 示例 |
|-----|------|------|
| 悬念钩 | 话说一半不说 | "你以为他是谁？" |
| 转折钩 | 出乎意料的回答 | "我不走。" |
| 冲突钩 | 引发新的矛盾 | "那他呢？你不管他了？" |
| 情感钩 | 情感的微妙转折 | "……随你。" |

### 示例

```
【悬念钩】
"有些事，我现在不能告诉你。"
"什么事？"
他没答，只是看了她一眼。
那一眼里有太多她读不懂的东西。

【转折钩】
"你走不走？"
"不走。"
他愣了一瞬。
"……你说什么？"
"我说，"她看着他，一字一顿，"我不走。"

【冲突钩】
"我可以带你离开这里。"
"那你皇兄呢？你不管他了？"
沉默。
长久的沉默。

【情感钩】
"……随你。"
就两个字。
可她听出了别的意思。
```

## 十、对话设计的"禁止"清单

### 绝对禁止

```
✗ 对话完全等于字面意思（没有潜台词）
✗ 两人对话完全理解对方（没有误会）
✗ 对话过于书面化（"我十分感谢您的好意"——古代人不会这么说）
✗ 对话过于冗长（超过5句没有动作描写）
✗ 所有角色说话风格一样（没有区分度）
✗ 用对话解释读者已经知道的信息（"你应该记得，你的母亲……"）
✗ 对话中出现AI高频句式（"值得注意的是""更重要的是"）
```

### 尽量避免

```
✗ 感叹号滥用（每千字不超过2个）
✗ "说"和"道"的机械替换
✗ 连续三句以上的纯对话（没有动作/心理穿插）
✗ 问句后立刻回答（缺少节奏/沉默）
```

## 十一、快速参考：对话设计自检

```
【每段对话自检】

□ 这句话有潜台词吗？
□ 这句话体现了谁的性格？
□ 这句话和上一句的节奏有变化吗？
□ 这句话有没有给下一句留钩子？
□ 说话人的情绪状态清晰吗？

【心声+言语配合自检】

□ 心声和言语是反的吗？（如果不是，有意为之吗？）
□ 心声是否揭示了言语背后的真正含义？
□ 读者能读出潜台词吗？

【古代言情特检】

□ 称谓是否正确？
□ 是否有身份感的体现？
□ 是否避免了现代用语？
```
