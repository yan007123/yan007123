# 爆款选题与卖点：市场导向指南

## 核心真理

**写得好不如选得好，选得好不如卖得好！**

- 选题 = 方向（决定能不能爆）
- 卖点 = 包装（决定读者看不看）
- 市场 = 验证（决定有没有流量）

**我们的目标：选对市场+精准包装=爆款！**

经过市场验证、读者百看不厌的核心设定。

### 女频10大经典梗

| 排名 | 经典梗 | 核心吸引力 | 代表设定 |
|------|--------|------------|----------|
| 1 | 霸总爱上我 | 被偏爱、被保护 | 冷酷总裁只对她温柔 |
| 2 | 重生复仇 | 后悔药、逆袭 | 回到过去改变命运 |
| 3 | 先婚后爱 | 禁忌感、日久生情 | 被迫结婚→真香 |
| 4 | 马甲文 | 掉马爽感、隐藏实力 | 多重身份层层揭露 |
| 5 | 替身文学 | 虐恋情深、身份错位 | 她是谁的替身？ |
| 6 | 追妻火葬场 | 虐后更甜、后悔莫及 | 失去后才珍惜 |
| 7 | 破镜重圆 | 遗憾弥补、深情不变 | 分开多年再相遇 |
| 8 | 双向暗恋 | 甜蜜酸涩、心有灵犀 | 互相喜欢都不知道 |
| 9 | 欢喜冤家 | 轻松搞笑、斗嘴生情 | 从互怼到相爱 |
| 10 | 救赎文 | 温暖治愈、互相拯救 | 他是她的光 |

### 微创新方法

**公式：经典梗 + 新元素 = 爆款**

#### 方法1：换背景
```
经典：霸总爱上我（现代）
创新：霸总爱上我（古代/星际/末世）
```

#### 方法2：换人设
```
经典：霸总男主 + 小白花女主
创新：霸总男主 + 马甲大佬女主
创新：病娇男主 + 直球女主
```

#### 方法3：加设定
```
经典：先婚后爱
创新：先婚后爱 + 读心术（能听到对方心声）
创新：先婚后爱 + 系统任务（必须秀恩爱）
```

#### 方法4：反转套路
```
经典：女主重生复仇
创新：男主重生追妻（男重生文）
经典：霸总护短
创新：女主护短霸总（女强男弱）
```

## 第三章：核心卖点提炼

### 什么是一句话卖点？

用一句话让读者产生"我想看"的冲动。

### 卖点公式

**【主角身份】+【核心冲突】+【独特设定】+【情感承诺】**

**示例**：
```
《重生后我成了影帝的白月光》
卖点：三线小明星重生回到18岁，
      发现未来的影帝现在还是个小透明，
      而她知道他会成为巨星——
      于是她开始投资他、培养他、撩拨他，
      却不知道他早就暗恋她多年。
```

### 卖点检查清单

- [ ] 主角有记忆点吗？（身份/特点）
- [ ] 冲突够吸引人吗？（问题/阻碍）
- [ ] 设定够新颖吗？（差异化）
- [ ] 情感够强烈吗？（甜/虐/爽）
- [ ] 一句话能说清楚吗？

## 第四章：书名设计——第一眼生死战

### 书名的作用

1. 告诉读者这是什么类型
2. 激发好奇心
3. 让读者记住

### 爆款书名公式

#### 公式1：核心梗直接型（最保险）
```
《重生之XXXX》
《总裁的XXXX》
《XXXX的替嫁新娘》
```

**适用**：新手、追求稳定流量
**优点**：读者一眼知道是什么
**缺点**：同质化严重

#### 公式2：悬念好奇型（最勾人）
```
《离婚后，前夫每天求复合》
《闪婚后发现老公是隐藏大佬》
《重生后我成了死对头的心尖宠》
```

**适用**：有一定写作经验
**优点**：好奇心强，点击率高
**缺点**：内容要对得起书名

#### 公式3：人设反差型（最吸睛）
```
《冷酷总裁的甜心小娇妻》
《病娇王爷的直球王妃》
《高冷影帝的追星小迷妹》
```

**适用**：人设鲜明的文
**优点**：CP感强
**缺点**：容易脸谱化

#### 公式4：情绪冲击型（最抓心）
```
《蚀骨危情：总裁的替身前妻》
《爱你成瘾：偏执总裁的囚宠》
《一念成殇：重生后我绝不原谅》
```

**适用**：虐文、强情绪文
**优点**：情绪冲击力强
**缺点**：容易劝退不喜欢虐的读者

### 书名禁忌

❌ **不要太文艺**
- 错误：《流年》《彼岸花开》《岁月静好》
- 问题：读者不知道是什么类型

❌ **不要太长**
- 错误：《重生后我成了影帝的白月光之霸道总裁爱上我》
- 问题：记不住，显示不全

❌ **不要欺骗**
- 书名是甜宠，内容是虐文
- 问题：读者愤怒，差评如潮

❌ **不要抄袭**
- 和爆款书名太像
- 问题：被认为是山寨，掉价

### 书名测试方法

1. **A/B测试**：想3个书名，让朋友选
2. **目标读者测试**：给目标读者看，问他们会不会点
3. **平台测试**：有些平台可以改书名，测试哪个点击率高

## 第五章：简介设计——第二眼生死战

### 简介的作用

1. 扩展书名的信息
2. 展示核心卖点
3. 建立期待感

### 爆款简介公式

#### 结构1：三段式（最经典）
```
【背景/人设】（1段）
【核心冲突】（1段）
【悬念/承诺】（1段）
```

**示例**：
```
林晚重生了，回到18岁那年。
上一世，她被渣男欺骗，被闺蜜背叛，最后惨死街头。
这一世，她要擦亮眼睛，让那些伤害她的人付出代价。

只是，那个未来的商业帝国掌权人，
现在为什么还是个小可怜？
她记得，他后来会成为叱咤风云的大佬。
于是她开始投资他、培养他、顺便撩拨他。

可她不知道的是，
这个"小可怜"早就暗恋她多年，
一直在等她回头看他一眼。
【甜宠+重生+双向暗恋】
```

#### 结构2：对话式（最生动）
```
【吸睛对话】（1句）
【背景展开】（2-3句）
【核心卖点】（1句）
```

**示例**：
```
"离婚吧。"
"好，财产对半，孩子归我。"
顾沉舟愣住了，他以为她会求他。

林晚潇洒签字，转身就走。
前世她爱他如命，最后换来的是背叛和死亡。
这一世，她只想为自己而活。

只是，那个曾经对她冷漠的男人，
为什么开始每天堵她的门？
"晚晚，我错了，再给我一次机会好不好？"
"不好，滚。"
【虐渣+追妻火葬场+爽文】
```

#### 结构3：悬念式（最勾人）
```
【抛出问题】（1句）
【背景铺垫】（2-3句）
【不揭示答案】（留下悬念）
```

**示例**：
```
所有人都知道，沈知意爱顾沉舟爱得死去活来。
可没人知道，她接近他，只是为了复仇。

三年前，她家破人亡，幕后黑手就是顾家。
她隐姓埋名，整容换脸，成为他的未婚妻。
只为在他最信任她的时候，给他致命一击。

只是，当真相揭露的那一刻，
为什么心会这么痛？
而他看她的眼神，为什么不是愤怒，而是……心疼？
【虐恋情深+复仇+误会】
```

### 简介检查清单

- [ ] 前50字有钩子吗？（读者会停下来吗？）
- [ ] 主角清晰吗？（读者知道谁是主角吗？）
- [ ] 冲突明确吗？（读者知道问题是什么吗？）
- [ ] 卖点展示了吗？（读者知道这是什么类型吗？）
- [ ] 有让人想点"开始阅读"的冲动吗？

## 第六章：标签与关键词

### 标签的作用

1. 精准推荐（平台根据标签推给目标读者）
2. 读者筛选（读者根据标签找书）
3. 搜索优化（关键词搜索）

### 标签选择原则

**必须标签**（2-3个）：
- 主类型：现言/古言/幻言/悬疑
- 子类型：甜宠/虐恋/爽文/重生
- 核心梗：霸总/重生/马甲/先婚后爱

**辅助标签**（2-3个）：
- 人设：女强/病娇/高冷/腹黑
- 元素：娱乐圈/豪门/校园/宫斗
- 情绪：轻松/正剧/暗黑

### 热门标签组合

| 类型 | 推荐标签组合 |
|------|--------------|
| 现言甜宠 | 现言+甜宠+霸总+轻松+豪门 |
| 古言重生 | 古言+重生+复仇+爽文+女强 |
| 马甲文 | 现言+马甲+掉马+爽文+甜宠 |
| 先婚后爱 | 现言+先婚后爱+甜宠+轻松+豪门 |
| 虐恋情深 | 现言/古言+虐恋+追妻+正剧+豪门 |
| 娱乐圈 | 现言+娱乐圈+甜宠+爽文+马甲 |

## 第七章：竞品分析

### 为什么要做竞品分析？

1. 了解市场趋势
2. 学习成功案例
3. 找到差异化空间
4. 避免同质化

### 竞品分析模板

```
【书名】：《XXX》
【作者】：XXX
【类型】：现言/古言/幻言
【核心梗】：重生/霸总/马甲
【一句话卖点】：
【优点】：
- 1. 
- 2. 
- 3. 
【缺点/可改进】：
- 1. 
- 2. 
【我们可以学习的】：
【我们可以差异化的】：
```

### 分析维度

1. **书名**：为什么吸引点击？
2. **简介**：怎么建立期待的？
3. **开篇**：怎么抓住读者的？
4. **人设**：主角有什么特点？
5. **节奏**：爽点和虐点怎么分布的？
6. **评论区**：读者喜欢什么？吐槽什么？

## 第八章：选题决策流程

### 步骤1：确定方向（1天）

**问题清单**：
1. 我想写什么类型？（现言/古言/幻言）
2. 我想写什么情绪？（甜/虐/爽）
3. 我擅长写什么？（情感/剧情/人设）
4. 当前市场什么火？（看榜单）

**输出**：2-3个备选方向

### 步骤2：市场调研（2-3天）

**操作**：
1. 看平台榜单（新书榜、热销榜）
2. 分析Top10的共同点
3. 找3-5本竞品深入分析
4. 看评论区了解读者需求

**输出**：市场机会点

### 步骤3：确定选题（1天）

**决策矩阵**：

| 选题 | 市场热度 | 差异化 | 个人能力 | 综合评分 |
|------|----------|--------|----------|----------|
| A | ⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ | 11 |
| B | ⭐⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ | 10 |
| C | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 12 |

**选择**：综合评分最高的

### 步骤4：包装测试（1-2天）

**操作**：
1. 设计3个书名
2. 写3版简介
3. 找目标读者测试
4. 选择点击率最高的

### 步骤5：开始写作

**记住**：选题和包装占成功的50%，写作占50%！
