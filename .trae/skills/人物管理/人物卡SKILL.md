# 人物卡 SKILL

> 该SKILL用于系统化管理小说中的所有人物，确保人物一致性。
> **触发条件**：用户要求"查看人物"、"添加人物"、"更新人物"、"人物关系"时执行。

---

## 核心目标

1. **统一人物信息来源**：所有人物信息集中在人物卡中
2. **追踪人物变化**：记录人物关系、性格、外貌的变化
3. **确保一致性**：新章节创作时自动对照人物卡
4. **辅助伏笔管理**：人物相关的伏笔与人物卡关联

---

## 功能模块

---

### 模块一：查看人物

**命令**：`查看人物` / `人物列表` / `都有谁出场了`

**操作**：
1. 读取 `人物卡库.md`
2. 按重要性分类展示（核心人物/次要人物/路人）
3. 标注最近出场的人物

**输出格式**：
```
【人物列表 · 共X人】

⭐ 核心人物（X人）：
1. 沈雾 — 女频主角，和离女，拥有同心蛊能力
2. 萧渊 — 废太子，表面冷漠内心保护
3. 青黛 — 沈雾丫鬟，忠心，发现有人监视

👤 次要人物（X人）：
4. 赵虎 — 押送官，暂时允许随行
5. 柳氏 — 第X章出场，身份待揭露

👥 路人（X人）：
6-10. 省略

最近出场：
- 第4章：沈雾、萧渊、青黛、赵虎
```

---

### 模块二：查看单个人物详情

**命令**：`查看[人物名]` / `[人物名]是什么设定`

**操作**：
1. 在人物卡库中搜索该人物
2. 显示完整人物卡

**输出格式**：
```
【人物卡：萧渊】

⭐ 核心人物 | 出场：第1-4章（4次）

基本信息：
- 姓名：萧渊
- 性别：男
- 年龄：27岁
- 身份：废太子，流放中
- 当前位置：流放路（黔州方向）

外貌特征：
- 身高：约185cm，体型清瘦但有力
- 肤色：苍白（常年幽禁）
- 眼睛：深邃，情绪难辨
- 标志性特征：右眼下有旧伤疤

性格特征：
- 核心：外冷内热，隐忍
- 习惯：沉默，说话简短
- 口头禅："不必"、"随你"
- 隐藏面：内心对沈雾有保护欲，母仇未报

心声特征：
- 内心：关怀但不说出口
- 常用比喻："冰下暗流"
- 触发：涉及沈雾安全时

关系：
- 沈雾：单向心声感知，想保护
- 青黛：允许跟随
- 赵虎：表面应付

⚠️ 一致性提醒：
- 保持"沉默寡言"风格
- 避免过度展露内心柔软
```

### 模块三：添加新人物

**命令**：`添加人物` / `新人物` / `有个新角色`

**操作**：
1. 询问基本信息
2. 创建人物卡
3. 自动分配ID（如P007）
4. 更新人物卡库

**交互流程**：
```
用户：添加一个新人物，是个大夫
AI：
请提供：
1. 姓名：____________
2. 性别：____________
3. 年龄：____________
4. 与主角关系：____________
5. 标志性特征：____________

用户：姓周，是个游医，帮过沈雾
AI：
✅ 已创建人物卡

【人物卡：周大夫】
ID：P007
姓名：周游医（名字待补充）
性别：男
年龄：约40岁
身份：游医
与主角关系：曾帮助过沈雾
标志性特征：药箱不离身

⚠️ 待完善：
- 名字待补充
- 出场章节待记录
- 是否成为伏笔相关人物？

```

---

### 模块四：更新人物信息

**命令**：`更新人物` / `[人物名]变了`

**触发时机**：
- 人物关系发生变化
- 人物外貌有变化
- 人物性格有新展现
- 人物新技能/能力出现

**操作**：
1. 确认人物
2. 确认更新内容
3. 在人物卡中添加"变更记录"

**输出格式**：
```
✅ 【人物卡已更新】

人物：青黛
更新内容：
- 技能：新增"会点拳脚"（幼年流浪所学）
- 出场：第5章将首次展示

变更记录：
[2026-04-15] 新增技能设定（拳脚）
```

## 人物卡检查清单

### 创建新人物时
- [ ] 姓名独特，不与现有人物混淆
- [ ] 性格标签明确（不超过3个核心词）
- [ ] 与现有人物的关系清晰
- [ ] 标志性特征具体（可识别）
- [ ] 出场目的明确（为什么需要这个人物）

### 创作中检查
- [ ] 对话风格符合人物设定
- [ ] 行为符合性格
- [ ] 外貌描写与人物卡一致
- [ ] 没有无意中美化/丑化人物

### 人物变化时
- [ ] 记录变更原因
- [ ] 确保变化有铺垫
- [ ] 更新人物卡
- [ ] 通知大纲管理SKILL更新伏笔
