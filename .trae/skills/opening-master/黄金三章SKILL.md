# 开篇黄金三章：生死之战

## 残酷真相

**女频读者3章定生死：**
- 第1章：看不进去 → 直接划走
- 第2章：没有期待 → 加入书架但不看
- 第3章：不想追更 → 永久流失

**编辑3章定签约：**
- 第1章：有没有卖点
- 第2章：会不会写
- 第3章：值不值得培养

**我们的目标：让读者3章之内必须追更！**

---

## 第一章：第一章——黄金300字

### 核心任务

**300字内必须完成：**
1. 抓住眼球（让读者停下来）
2. 建立好奇（让读者想知道）
3. 设定基调（让读者知道这是什么类型）

### 7大黄金开局模式

#### 模式1：危机开局（最保险）
**公式**：主角陷入危机 + 悬念

**示例**：
```
刀架在脖子上的时候，林晚还在想——
她到底是哪里露出了马脚？
三天前她穿越到这本书里，
成了那个活不过三章的炮灰女配。
而现在，正是第三章。
```

**适用**：穿越、重生、逆袭文
**效果**：紧张感+代入感+悬念

---

#### 模式2：反差开局（最吸睛）
**公式**：表面A + 实际B（巨大反差）

**示例**：
```
全京城都知道，丞相府的嫡女是个傻子。
直到那天宫宴上，
她一首《将进酒》惊艳四座，
顺便把挑衅她的公主怼得哑口无言。
众人这才发现——
这哪是傻子？分明是扮猪吃老虎！
```

**适用**：打脸、逆袭、马甲文
**效果**：爽感+期待感

---

#### 模式3：暧昧开局（女频专用）
**公式**：亲密接触 + 身份悬念

**示例**：
```
沈知意醒来的时候，发现自己在一个陌生男人怀里。
更可怕的是——
她身上只穿了一件衬衫，还是他的。
男人睁开眼，声音沙哑："醒了？"
沈知意："......我们昨晚？"
男人："你说呢？"
沈知意看着满地的衣服，陷入了沉思。
她连这男人是谁都不知道！
```

**适用**：现言、甜宠、先婚后爱
**效果**：脸红心跳+强烈好奇

---

#### 模式4：冲突开局（最直接）
**公式**：直接冲突 + 主角态度

**示例**：
```
"离婚吧。"
顾沉舟把协议书扔在桌上，语气平淡得像在讨论天气。
林晚笑了，拿起协议书看都没看就签了字。
"好啊，财产对半，孩子归我。"
顾沉舟皱眉："你——"
"怎么？"林晚站起身，居高临下地看着他，
"顾总不会以为，我林晚离了你活不下去吧？"
```

**适用**：虐恋、复仇、离婚逆袭
**效果**：情绪冲击+人设建立

---

#### 模式5：悬念开局（最勾人）
**公式**：神秘事件 + 主角卷入

**示例**：
```
那封信送到我手上的时候，
送信人已经死了。
我打开信封，里面只有一张照片——
照片上的女孩，和我长得一模一样。
而照片背面写着一行字：
"不要相信任何人，包括你自己。"
```

**适用**：悬疑、推理、灵异
**效果**：强烈好奇+紧张感

---

#### 模式6：日常开局（最温馨）
**公式**：有趣日常 + 小冲突

**示例**：
```
"老板，来碗牛肉面，多加牛肉！"
"姑娘，你已经连续吃了一个月了，
  不腻吗？"
"不腻！"我拍着胸脯，
"只要能看到你儿子，吃一年都不腻！"
老板儿子从后厨探出头，脸红了。
```

**适用**：甜宠、美食、种田文
**效果**：轻松愉快+CP感

---

#### 模式7：重生/穿越开局（最套路但有效）
**公式**：死亡/穿越瞬间 + 新身份认知

**示例**：
```
上一秒，我还在颁奖典礼上被吊灯砸死。
下一秒，我成了古代一个即将被赐死的妃子。
看着面前端着毒酒的太监，
我深吸一口气——
没事，我可是拿过奥斯卡影后的人。
演，就完事了。
```

**适用**：重生、穿越、宫斗
**效果**：代入感+金手指期待

### 第一章检查清单（致命级）

写完第一章，必须检查：

- [ ] **前300字有钩子吗？**（读者停下来了吗？）
- [ ] **主角出场了吗？**（读者知道谁是主角了吗？）
- [ ] **主角有记忆点吗？**（性格/处境/特点）
- [ ] **冲突/悬念出现了吗？**（读者想知道后续吗？）
- [ ] **类型明确了吗？**（读者知道这是什么文吗？）
- [ ] **有情绪冲击吗？**（爽/甜/虐/紧张？）
- [ ] **想让人点第二章吗？**

**如果有任何一个没做到，重写！**

## 第二章：第二章——建立期待

### 核心任务

**让读者产生"我想知道接下来会发生什么"的期待**

### 第二章必须完成

1. **展开第一章的钩子**
   - 不能吊着不解决，但也不能完全解决
   - 解决一个小问题，抛出更大的问题

2. **加深人物印象**
   - 主角的更多侧面
   - 重要配角出场
   - 人物关系建立

3. **建立主线方向**
   - 主角的目标是什么？
   - 阻碍是什么？
   - 读者能预期到什么？

### 第二章结构公式

```
【承接】回应第一章悬念（30%）
    ↓
【发展】主角行动/反应（50%）
    ↓
【新钩子】抛出更大悬念（20%）
```

### 第二章检查清单

- [ ] 第一章的钩子有回应吗？
- [ ] 主角的目标清晰了吗？
- [ ] 重要配角出场了吗？
- [ ] 有新的悬念/冲突吗？
- [ ] 读者更了解主角了吗？
- [ ] 想让人点第三章吗？

---

## 第三章：第三章——锁定追更

### 核心任务

**让读者做出"我要追这本书"的决定**

### 第三章必须完成

1. **展示故事潜力**
   - 这不是一篇短文，有长篇潜力
   - 有更多人物、更多冲突、更多悬念

2. **建立情感连接**
   - 让读者喜欢/同情/支持主角
   - 让读者期待主角的成功

3. **埋下追更钩子**
   - 一个必须追下去的理由
   - 可以是剧情悬念，也可以是感情期待

### 第三章钩子类型

#### 剧情型钩子
```
主角发现了一个秘密，但只透露了一半
新的反派/危机出现
主角做出了一个重要决定
```

#### 感情型钩子（女频最重要）
```
男女主有了第一次亲密接触
男主对女主表现出特殊态度
女主对男主产生了好奇/好感
一个关于感情的悬念（比如男主的秘密）
```

#### 成长型钩子
```
主角获得了金手指/能力
主角决定不再忍耐，要开始反击
主角遇到了改变命运的机会
```

### 第三章检查清单（生死级）

- [ ] 读者喜欢主角了吗？
- [ ] 读者期待主角成功吗？
- [ ] 故事有长期潜力吗？
- [ ] 有必须追更的理由吗？
- [ ] 感情线有进展/期待吗？（女频）
- [ ] **读者会点"下一章"吗？**

## 开篇致命错误（绝对不能犯）

### ❌ 错误1：大段背景介绍
**症状**：第一章全是世界观、家族关系、前世今生
**后果**：读者直接划走
**解决**：背景分散插入，先上冲突

### ❌ 错误2：人物太多
**症状**：第一章出场10个人，读者记不住
**后果**：混乱，不知道谁是主角
**解决**：主角先出场，其他人分批介绍

### ❌ 错误3：太平淡
**症状**：第一章就是日常，没有冲突
**后果**：读者觉得无聊
**解决**：第一章必须有冲突/悬念/钩子

### ❌ 错误4：节奏太慢
**症状**：铺垫太多，迟迟不进正题
**后果**：读者失去耐心
**解决**：前3章必须有一个小高潮

### ❌ 错误5：主角不讨喜
**症状**：主角太弱/太蠢/太坏
**后果**：读者无法代入
**解决**：给主角一个让人喜欢的点

### ❌ 错误6：类型不清
**症状**：读者看完3章不知道这是什么类型
**后果**：无法精准推荐
**解决**：前3章明确展示类型特色

## 不同类型开篇要点

### 古言甜宠
- 第一章：相遇或特殊互动
- 第二章：展示男主特殊态度
- 第三章：感情萌芽或外部阻碍

### 现言霸总
- 第一章：意外相遇/亲密接触
- 第二章：身份揭晓/再次交集
- 第三章：男主开始追妻/护短

### 重生复仇
- 第一章：重生瞬间+前世回忆（简短）
- 第二章：开始布局/第一次打脸
- 第三章：复仇计划展开+新线索

### 穿越逆袭
- 第一章：穿越+了解处境
- 第二章：展示能力/第一次出彩
- 第三章：引起注意+新的挑战

### 虐恋情深
- 第一章：误会/伤害发生
- 第二章：展示深情+误会加深
- 第三章：虐点升级+希望微光

### 开篇修改流程

1. 写完后放24小时
2. 用新鲜的眼光读一遍
3. 对照检查表逐项检查
4. 找beta读者试读（如果可能）
5. 根据反馈修改
6. 重复直到满意
