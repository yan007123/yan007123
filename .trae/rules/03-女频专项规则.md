# 女频专项规则

## 核心原则

**情感线为主，细腻为先，推拉为美，甜虐交织。**

---

## 一、情感节奏控制

### 1.1 甜虐比控制

```yaml
风格设定:
  正剧向: "甜:虐 = 6:4"
  
当前项目: "正剧向(甜:虐=6:4)"
```

### 1.2 连续章节限制

| 类型 | 最大连续章数 | 说明 |
|------|-------------|------|
| 甜章 | ≤3章 | 连续甜会腻，需要虐调节 |
| 虐章 | ≤2章 | 连续虐会弃，需要甜救赎 |
| 平淡章 | ≤2章 | 信息释放章节，不能太长 |

### 1.3 情感递进节奏

```
标准情感线:
心动 → 误解 → 靠近 → 障碍 → 试探 → 确认 → 新障碍 → 突破 → 升华

当前进度(第5章):
沈雾: 心动初期(30%)
萧渊: 深爱但克制(80%)

第5-10章目标:
- 第5章: 心动加深(靠肩而眠)
- 第6章: 靠近(共同遇险)
- 第7章: 了解(知晓过去)
- 第8章: 障碍(身份危机)
- 第9章: 试探(影七现身)
- 第10章: 确认(配合默契)
```

### 1.4 每章情感标签

**写章前先贴标签**：
- 💕 甜章（感情升温）
- 💔 虐章（误会/阻碍）
- ⚔️ 冲突章（外部矛盾）
- 🧠 智斗章（信息战）
- 📖 过渡章（信息释放）

**标签使用规则**：
- 连续3章💕后，必须来1章💔或⚔️
- 连续2章💔后，必须来1章💕
- 🧠章前后需要📖章缓冲

## 二、章节钩子设计系统

### 2.1 钩子类型库

| 类型 | 示例 | 效果 | 适用场景 |
|------|------|------|---------|
| **悬念型** | "门外传来脚步声，是谁？" | 制造紧张 | 任何场景 |
| **情感型** | "他心声说：'别离开我'" | 打动读者 | 感情戏 |
| **冲突型** | "大军已经兵临城下" | 制造紧迫 | 战争/对抗 |
| **反转型** | "原来他一直知道" | 制造震撼 | 真相揭露 |
| **预言型** | "三天后，她后悔了" | 制造期待 | 过渡章节 |
| **定格型** | "他低头，吻上了她的..." | 制造遐想 | 亲密戏前 |

### 2.2 钩子强度评级

★☆☆☆☆: "轻微好奇，可看可不看"
★★☆☆☆: "有点想知道，大概率会点"
★★★☆☆: "很想知道，一定会点"
★★★★☆: "抓心挠肝，必须马上看"
★★★★★: "神级钩子，不点睡不着"

每章目标: "≥★★★☆☆"
高潮章: "≥★★★★☆"
```

### 2.3 钩子位置

**必须在章末最后100字内**

```
错误示例：
...（中间内容）...
他转身离去。沈雾看着他的背影，心中五味杂陈。
（后面还有500字环境描写）

正确示例：
...（中间内容）...
他转身离去。沈雾看着他的背影，心中五味杂陈。

门外突然传来脚步声——

不是他的。

是谁？
```

### 2.4 钩子设计检查表

写章末前自检：
- [ ] 最后100字有钩子吗？
- [ ] 钩子强度≥★★★☆☆吗？
- [ ] 读者会想知道"接下来呢"吗？
- [ ] 钩子类型与章节标签匹配吗？

## 三、五感描写检查

### 3.1 基础要求

**每章必须包含**：
- 视觉描写：≥3处
- 听觉描写：≥2处
- 嗅觉/触觉：≥1处

**情感场景必须**：
- 五感全开
- 环境氛围铺垫：≥100字

### 3.2 五感与情感对应

| 感官 | 情感暗示 | 示例 |
|------|---------|------|
| **视觉** | 距离/温度 | "他站在三步之外，眉眼含霜" |
| **听觉** | 紧张/放松 | "他的呼吸声在寂静中格外清晰" |
| **嗅觉** | 亲密/回忆 | "他身上有很淡的雪松味道" |
| **触觉** | 真实/渴望 | "他的手指擦过她的手背，温热" |
| **味觉** | 甜蜜/苦涩 | "那杯茶尝起来是苦的" |

### 3.3 五感检查表

写情感场景前自检：
- [ ] 环境氛围铺垫≥100字了吗？
- [ ] 视觉描写有了吗？（看到了什么）
- [ ] 听觉描写有了吗？（听到了什么）
- [ ] 嗅觉/触觉有了吗？（闻到/摸到什么）
- [ ] 五感与当前情感匹配吗？

## 四、推拉美学规则

### 4.1 推拉定义

```yaml
近:
  身体: "靠近、触碰、同处一室"
  语言: "关心、温柔、说心里话"
  眼神: "注视、深情、不舍"
  
退:
  理性: "身份差距、责任束缚"
  误会: "信息不对称、第三人"
  身份: "立场对立、使命不同"
  责任: "家族、国家、承诺"
```

### 4.2 推拉原则

**核心原则**：
1. 每次"推"必须制造下一次"拉"的理由
2. 每次"拉"必须留下下一次"推"的缝隙
3. 绝不让两人在第一次心动后就毫无障碍地在一起

**示例**：
```
第5章（拉）: 靠肩而眠，身体靠近
  ↓ 留下缝隙：萧渊心声"要克制"
  
第6章（推）: 山间遇险，萧渊为救她受伤
  ↓ 制造理由：沈雾心疼，主动照顾
  
第7章（拉）: 照顾中了解他的过去，情感靠近
  ↓ 留下缝隙：沈雾发现他的秘密（内功受损）
  
第8章（推）: 身份危机，苗疆血统被质疑
  ↓ 制造理由：萧渊保护她，两人共同面对
```

### 4.3 推拉节奏表

| 章节 | 推拉 | 具体表现 | 留下的缝隙 |
|------|------|---------|-----------|
| 5 | 拉 | 靠肩而眠 | 萧渊克制 |
| 6 | 推 | 遇险受伤 | 沈雾心疼 |
| 7 | 拉 | 了解过去 | 发现秘密 |
| 8 | 推 | 身份危机 | 共同面对 |
| 9 | 拉 | 影七现身 | 信任考验 |
| 10 | 拉 | 配合默契 | 但未来未知 |

## 五、心声描写专项

### 5.1 心声使用原则

```yaml
触发条件:
  - "萧渊情绪波动时自动触发"
  - "沈雾无法控制开关"
  
内容特点:
  - "真实想法，无法伪装"
  - "话痨型，与外表反差"
  - "越克制，心声越滚烫"
  
使用频率:
  - "每章至少3处心声"
  - "情感场景增加密度"
  - "信息释放场景减少"
```

### 5.2 心声类型

| 类型 | 示例 | 效果 |
|------|------|------|
| **关心型** | "她冷吗？该死，我应该把外套给她" | 展现温柔 |
| **克制型** | "冷静，萧渊，冷静，你不能想这些" | 展现挣扎 |
| **占有欲** | "她是我的，只能是我的" | 展现病娇 |
| **脆弱型** | "如果她走了，我会疯的" | 展现脆弱 |
| **吐槽型** | "这个老狐狸，真当我是傻子？" | 展现真实 |

### 5.3 心声与反应对照

```
萧渊心声: "她好香……不行，要克制……"
  ↓
沈雾反应: 装睡，心跳加速，耳尖泛红
  ↓
读者知道: 两人都在装，暧昧升级
  ↓
戏剧张力: 信息差制造的期待感
```

## 六、情感具象化规则

### 6.1 禁止直接命名情感

❌ **错误**：
- "她感到很悲伤"
- "他愤怒极了"
- "她心动了"

✅ **正确**：
- "她把他的衬衫叠了又叠。叠了三遍。第三遍的时候她发现自己在叠的方向不对——他习惯领口朝左，她总是忘。"
- "他把烟掐灭在手背上。疼了一下。不够。又点了一根。"
- "他在她旁边站定的时候，她突然忘了接下来要说什么。明明准备了一整天的话。"

### 6.2 情感-行为对照表

| 情感 | 不要写 | 要写 |
|------|--------|------|
| 悲伤 | 她哭了 | 她把他的衬衫叠了又叠，叠错了方向 |
| 愤怒 | 他很生气 | 他把烟掐灭在手背上 |
| 心动 | 她心动了 | 她突然忘了接下来要说什么 |
| 恐惧 | 她很害怕 | 她把门反锁了，又推了柜子顶住 |
| 嫉妒 | 她嫉妒了 | 她笑着，笑得客气又完整，在场所有人都觉得冷 |
