# 质检标准规则

## 核心原则

**不是找茬，是确保质量；不是完美，是避免低级错误。**

---

## 一、去AI味质检

### 1.1 句式多样性检查

**标准**：禁止连续3个句子使用相同句式结构

**常见重复句式**：
- ❌ "她……。她……。她……。"（连续主语相同）
- ❌ "他看着她。他想着她。他保护她。"（连续主谓宾）
- ❌ "突然……。突然……。"（连续副词开头）

**修改方案**：
```
原文:
她看着窗外。她想着心事。她叹了口气。

修改:
窗外的雪下得更大了。（环境切入）
她盯着那片白，思绪飘远了。（主语后置）
良久，一声轻叹落在寂静里。（省略主语）
```

**检查方法**：
- 任意连续3句话，句式结构必须不同
- 接受度：每章≤3处轻微重复

---

### 1.2 词汇离散度检查

**标准**：同一概念在800字范围内必须使用至少3种不同表达

**示例**：
```
概念：愤怒

第1次：火气直窜天灵盖
第2次：脸黑得像锅底  
第3次：手都在抖，指甲掐进肉里

❌ 错误（重复）:
他很愤怒。他愤怒极了。他太愤怒了。
```

**高频概念替换表**：

| 概念 | 表达1 | 表达2 | 表达3 |
|------|-------|-------|-------|
| 愤怒 | 火气直窜 | 脸黑如锅底 | 手抖掐肉 |
| 心动 | 心跳漏拍 | 耳尖泛红 | 忘了要说什么 |
| 悲伤 | 叠衣服叠错 | 通讯录翻四次 | 盯着窗发呆 |
| 紧张 | 绞手指 | 呼吸急促 | 后背冒汗 |
| 开心 | 嘴角压不住 | 脚步轻快 | 看什么都顺眼 |

---

### 1.3 AI高频词黑名单

**绝对禁用**：
- 值得注意的是
- 此外
- 总而言之
- 具体来说
- 需要指出的是
- 显而易见
- 毫无疑问
- 从某种意义上说
- 综上所述
- 在这个过程中
- 与此同时（叙事中每万字≤1次）

**检查方法**：
- 全文扫描黑名单词汇
- 发现即标记，要求替换

---

### 1.4 节奏波动检查

**标准**：段落长度必须呈现不规则波动

**推荐节奏**：
```
长段落（150-300字）
  ↓
超短段落（8-30字）
  ↓
中段落（60-120字）
  ↓
短段落（20-50字）
  ↓
单字/单句（1-10字）
```

**示例**：
```
长段落：
她坐在窗边，看着窗外的雪。那是建安十四年的腊月初九，京城下了第一场雪。雪片很大，像鹅毛一样飘落，落在窗台上，落在院子里，落在她的心里。她想起三年前，也是这样一个雪天，她嫁入了太子府。

超短段落：
那时候，她以为那是幸福的开始。

中段落：
现在她知道，那只是一场梦。梦醒了，她还在原地，而他已经走远了。

短段落：
雪还在下。

单字：
冷。
```

---

### 1.5 语义毛刺检查

**标准**：每千字0-2处语义毛刺，制造新鲜感

**毛刺类型**：
- **矛盾修饰**：温柔的残忍、精确的混乱
- **通感错位**：那个声音是橙色的、他的沉默有重量
- **荒诞直觉**：她总觉得今天的太阳是假的

**检查方法**：
- 全文扫描非常规表达
- 确保每千字不超过2处
- 避免过度使用变成"文学腔"

---

### 1.6 微瑕疵注入

**标准**：每2000字必须包含至少3处微瑕疵

**可接受的瑕疵**：
- 轻微用词重复（50-100字内重复形容词）
- 时间/空间微小不一致
- 某个比喻用到一半换了一个
- 偶尔一句话写得过长

**示例**：
```
他像一头困兽——不，更像是被踩了尾巴的猫，炸毛的那种。
（比喻中途更换，人类写作特征）
```

## 二、一致性质检

### 2.1 人物一致性

**检查项**：
- [ ] 人物性格是否前后一致
- [ ] 人物能力是否前后一致
- [ ] 人物关系是否正确
- [ ] 人物称呼是否正确
- [ ] 人物知道的信息是否超出其视角

**OOC检测**：
```
萧渊性格：隐忍、克制、病娇（内心）

❌ OOC示例：
萧渊当众大喊："我爱沈雾！"

✅ 符合性格：
萧渊面无表情，心声狂乱："她是我的，只能是我的。"
```

### 2.2 设定一致性

**检查项**：
- [ ] 同心蛊规则是否一致
- [ ] 世界观设定是否一致
- [ ] 时间线是否正确
- [ ] 地点转换是否合理

**常见错误**：
- 第1章说玉坠是母后遗物，后面变成父皇遗物
- 第3章说黔州三千里，第5章说两千里
- 第2章说沈雾11岁母亲去世，第4章说12岁

### 2.3 伏笔一致性

**检查项**：
- [ ] 伏笔是否都有回收计划
- [ ] 回收时间是否合理（不能太近/太远）
- [ ] 伏笔内容是否前后一致

**检查方法**：
- 对照中期记忆-伏笔追踪表
- 确保每个"已埋设"都有对应的"回收"

## 三、女频专项质检

### 3.1 情感具象化检查

**标准**：禁止直接命名情感

**检查清单**：
- [ ] 没有"她感到悲伤"
- [ ] 没有"他愤怒极了"
- [ ] 没有"她心动了"
- [ ] 没有"他很高兴"

**替换检查**：
```
❌ 她感到很悲伤。
✅ 她把他的衬衫叠了又叠。叠了三遍。

❌ 他愤怒极了。
✅ 他把烟掐灭在手背上。疼了一下。不够。又点了一根。
```

### 3.2 五感描写检查

**标准**：
- 每章视觉≥3处
- 每章听觉≥2处
- 每章嗅觉/触觉≥1处
- 情感场景五感全开+环境铺垫≥100字

**检查方法**：
- 扫描关键词：看见、听到、闻到、摸到、尝到
- 统计每类感官出现次数

### 3.3 心声描写检查

**标准**：
- 每章心声≥3处
- 心声与外表反差
- 心声内容符合人物性格

**检查清单**：
- [ ] 萧渊外表冷漠时，心声是否滚烫
- [ ] 心声是否话痨型（符合设定）
- [ ] 心声是否真实（无法伪装）

### 3.4 钩子设计检查

**标准**：
- 每章有钩子
- 钩子在章末最后100字内
- 钩子强度≥★★★☆☆

**检查方法**：
- 查看章末100字
- 判断是否有悬念/情感冲击
- 评估读者是否会想点"下一章"

### 3.5 推拉节奏检查

**标准**：
- 连续甜章≤3章
- 连续虐章≤2章
- 每次"推"制造"拉"的理由
- 每次"拉"留下"推"的缝隙

**检查方法**：
- 对照中期记忆-爽点规划表
- 检查情感标签是否合理交替

## 四、质检执行流程

### 4.1 自动质检

**触发时机**：
- 章节生成完成后自动执行
- 用户说"检查"时执行

**执行顺序**：
```
1. 去AI味质检
   - 句式多样性
   - 词汇离散度
   - AI高频词黑名单
   - 节奏波动
   - 语义毛刺
   - 微瑕疵

2. 一致性质检
   - 人物一致性
   - 设定一致性
   - 伏笔一致性

3. 女频专项质检
   - 情感具象化
   - 五感描写
   - 心声描写
   - 钩子设计
   - 推拉节奏
```

### 4.2 质检报告格式

```
质检报告 - 第5章

去AI味检查:
✅ 句式多样性（无连续重复）
✅ 词汇离散度（800字内3种表达）
✅ AI高频词（未发现禁用词）
⚠️ 节奏波动（第3-5段长度相近，建议调整）
✅ 语义毛刺（2处，符合标准）
✅ 微瑕疵（3处，符合标准）

一致性检查:
✅ 人物一致性（萧渊性格符合）
✅ 设定一致性（同心蛊规则正确）
✅ 伏笔一致性（V004进展合理）

女频专项检查:
✅ 情感具象化（无直接命名）
⚠️ 五感描写（嗅觉描写较少，建议增加）
✅ 心声描写（5处，符合标准）
✅ 钩子设计（★★★★☆，位置正确）
✅ 推拉节奏（💕甜章，符合规划）

总体评价: 良好
建议修改:
1. 调整第3-5段长度，增加节奏波动
2. 增加1处嗅觉描写（如萧渊身上的味道）
```

### 4.3 自动修复

**可自动修复的问题**：
- AI高频词 → 自动替换为同义词
- 轻微句式重复 → 自动调整语序
- 标点符号 → 自动转换为中文标点

**需人工确认的问题**：
- 人物OOC
- 设定矛盾
- 伏笔冲突
- 情感节奏不合理
