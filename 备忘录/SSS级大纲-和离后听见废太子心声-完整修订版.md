# 《和离后，我听见了废太子的心声》完整大纲（修订版）

## 一、世界观设定

### 1.1 时代背景：大晟王朝

#### 政治格局

**皇权结构**：
- 先帝育有四子。大皇子萧渊为皇后谢氏所出，立为太子。
- 二皇子萧衍体弱多病，早年暴毙（实为萧承下毒）。
- 三皇子萧承为贵妃赵氏所出，外戚势力庞大，掌控吏部与刑部。
- 四皇子萧煜年仅六岁，生母早逝。

**权力更迭**：
先帝驾崩后，萧承以"太子失德"之名联合朝中势力废黜萧渊，扶持幼帝萧煜登基，自任监国摄政，实际掌权，待后期再废帝自己登基。萧渊被判流放黔州。

**朝堂势力三分**：
- **萧承一党**：吏部、刑部、部分禁军
- **谢家一脉**：兵部、西北边军，暗中支持萧渊
- **中立派**：户部、工部，观望摇摆

**沈家立场**：
沈家掌礼部，原依附太子党，萧渊被废后迅速切割，转向萧承示好。

#### 六部势力分布

| 部门 | 掌控者 | 职权 | 立场 |
|------|--------|------|------|
| 吏部 | 萧承 | 官员任免 | 反派阵营 |
| 户部 | 中立 | 财政大权 | 摇摆 |
| 礼部 | 沈家 | 礼仪祭祀 | 表面中立，实则摇摆 |
| 兵部 | 谢家 | 军权 | 支持萧渊 |
| 刑部 | 萧承 | 司法 | 反派阵营 |
| 工部 | 中立 | 工程营造 | 摇摆 |

#### 地理设定

- **京城**：大晟王朝都城，政治中心
- **黔州**：西南边陲，瘴气弥漫，环境恶劣
- **苗疆**：位于黔州以南，神秘部族聚集地

### 1.2 核心设定：同心蛊（SSS级创新设定）

#### 来源与传承

**起源**：苗疆巫蛊一脉，百年难遇的至情之蛊。非人为刻意施种，而是以特殊载体（含蛊者心头血浸润之物）长期贴身佩戴，蛊虫自行认主。

**前史**：
萧渊之母谢皇后出身谢家，而谢家祖上有一支与苗疆联姻的血脉。谢皇后临终前将家传玉簪（内封同心蛊虫）交予萧渊，言"将此簪赠予你此生最想守护之人，它会替母后看着你"。

**触发机制**：
- 萧渊在新婚之夜将玉簪赠予沈雾作为定情信物
- 蛊虫以沈雾的体温和心血为养，潜伏三年
- 和离之际，萧渊情绪剧烈波动（极度痛苦），触发了蛊虫的"心声传递"功能

#### 关键规则

| 规则 | 说明 |
|------|------|
| **单向传递** | 沈雾能听到萧渊心声，萧渊无法感知 |
| **自动触发** | 沈雾无法控制开关，萧渊情绪波动时自动触发 |
| **真实传递** | 蛊虫只传递"真实想法"，无法被伪装或欺骗 |

#### 距离限制

| 距离 | 效果强度 | 心声清晰度 | 情绪感知 |
|------|----------|------------|----------|
| 10米内 | 最强 | 清晰如耳边低语 | 强烈 |
| 100米内 | 中等 | 断断续续 | 明显 |
| 1000米内 | 微弱 | 只能感知情绪 | 微弱 |
| 超过1000米 | 无效 | 完全失效 | 无 |

#### 情绪共鸣

| 情绪类型 | 身体反应 | 描写方式 |
|----------|----------|----------|
| 愤怒 | 心跳加速、血液沸腾 | "她的心突然跳得厉害，像要撞破胸膛" |
| 恐惧 | 手脚冰凉、寒毛倒竖 | "一股寒意从脚底窜上来，她忍不住打了个寒颤" |
| 心动 | 脸红、心跳加速、身体发热 | "她的脸突然烫起来，心跳乱了节奏" |
| 悲伤 | 心痛、呼吸困难 | "胸口像被人攥着，疼得她喘不上气" |

#### 副作用（新增）

**精神损耗递进表**：

| 阶段 | 时间 | 症状 | 影响 |
|------|------|------|------|
| 初期 |  偶发性头痛 | 轻微不适 |
| 中期 | 耳鸣、失眠 | 影响日常 |
| 后期 | 记忆混淆（分不清哪些是自己的想法，哪些是萧渊的） | 精神损耗严重 |
| 临界 | 短暂失去自我意识 | 若不解蛊，将彻底沦为只能感知他人情绪的空壳 |

#### 解除方式

| 方式 | 条件 | 难度 |
|------|------|------|
| 蛊源者死亡 | 萧渊死则蛊消 | 极端 |
| 真心血解蛊 | 双方以真心血自愿解蛊，需苗疆圣女主持仪式 | 困难 |
| 载体持有者死亡 | 沈雾死则蛊消 | 极端 |

#### 创新价值

1. 在古言中引入"读心"概念，增强代入感
2. 距离限制增加策略性和戏剧性
3. 情绪共鸣机制强化情感表达
4. 副作用设定增加紧迫感和代价感

## 二、人物设定

### 2.1 女主：沈雾

#### 基本信息

- **身份**：礼部尚书嫡女→太子妃→和离弃妇→流放随行人→黔州女医→（后期）皇后
- **年龄**：19岁
- **外貌**：清冷素雅，右眼角有颗泪痣，身材纤细，手指修长适合针灸
- **标志性动作**：思考时轻抚泪痣、紧张时绞手指、专注时下唇轻咬

#### 核心性格

表面柔顺隐忍，内心极度清醒。她不是"傻白甜觉醒"式的女主，而是从一开始就看得清局势，只是在太子妃的位置上选择了隐忍。和离后的"觉醒"不是性格突变，而是不再需要隐忍。

#### 核心创伤

生母沈氏在她七岁时"病逝"，实际上是被继母长期克扣汤药、沈父冷落不管导致病情恶化而死。沈雾幼年目睹母亲凄凉死状，从此学会了"看人脸色、隐藏真实想法"的生存技能。她对"被抛弃"有极深恐惧，这也是她在太子府三年忍受冷遇却不离开的深层原因——她害怕自己一旦被抛弃就会像母亲一样死去。

#### 隐藏秘密（新增）

沈雾的生母并非普通官家女子，而是苗疆一支旁系血脉的后人（沈父当年娶她是看中了她家的一批珍贵药材资源，婚后对她毫无感情）。沈雾体内流着苗疆血脉，这是同心蛊能在她身上生效的根本原因，也是后期她能解蛊的关键。

#### 能力设定

| 能力 | 等级 | 来源 | 应用 |
|------|------|------|------|
| 医术 | 中等 | 生母所授+自学医书 | 针灸与草药辨识 |
| 观察力 | 极强 | 幼年生存技能 | 从细节推断真相 |
| 读心 | 核心 | 同心蛊 | 听到萧渊心声 |
| 智谋 | 中等 | 后期成长 | 主动利用心声进行信息战 |

#### 情感弧线

```
恐惧被抛弃 → 被迫独立 → 发现被爱 → 学会信任 → 主动去爱 → 直面信任危机 → 真正的灵魂平等
```

### 2.2 男主：萧渊

#### 基本信息

- **身份**：太子→废太子→流放犯→（暗线）暗中执棋者→（后期）夺回皇位
- **年龄**：24岁
- **外貌**：俊美冷峻，眉眼含霜，身材高大，手指修长，右肩有旧伤
- **标志性动作**：思考时摩挲腰间玉佩（母后遗物）、紧张时握紧拳头、看女主时眼神柔软

#### 核心性格

极致的隐忍与算计之下，藏着一颗被反复背叛后几近碎裂的心。他不是"深情男主顺便搞权谋"，而是"一个权谋家试图允许自己去爱"。

#### 核心创伤

1. **母后被毒杀**：八岁时亲眼看着母后被毒杀，先帝明知真相却为了朝堂平衡选择包庇赵贵妃一族，萧渊从此不再信任"亲情"
2. **柳氏背叛**：被废前夜发现侧妃柳氏是萧承安插的棋子，柳氏曾多次将他的部署泄露给萧承，导致他数次险些丧命，从此对"枕边人"产生极度不信任
3. **兄弟之死**：唯一信任过的兄弟萧衍被萧承毒杀，他未能阻止，深感愧疚

#### 对沈雾的复杂态度

- 三年太子妃期间，他不是对沈雾"冷漠"，而是刻意保持距离——因为每一个靠近他的人都可能是棋子，柳氏的背叛让他不敢再信任任何女人
- 但三年间他暗中观察沈雾的一切，从细节中确认她是真心实意的人，逐渐动了真心

#### 和离的真正原因（新增）

萧承在和离前三天向萧渊传话——"你的太子妃，沈家的嫡女，本王很感兴趣。你若不乖乖就范，本王不介意让她换个夫君。"萧渊知道萧承说到做到，和离是断了萧承以她为筹码的可能。

同时，萧渊暗中安排了后手——他知道沈家会抛弃沈雾，而沈雾走投无路后会被引导到他的暗棋路线上。也就是说，和离本身就是萧渊布局的一部分，他算到了沈雾会跟来，但他没算到的是——她是因为听到了心声才跟来的。这个"计划之外"才是真正的戏剧张力。

#### 被废的完整真相

萧渊并非完全被动被废。他在先帝病重期间发现了母后被毒杀的完整证据链，决定借"被废"之势暗中布局——被废后萧承会放松警惕，流放路上他可以摆脱京城的监视联络外部势力。但他没有料到萧承的手段比他想象的更狠——流放途中的追杀不是做样子，而是真的要取他性命。

#### 心声特质

萧渊的心声与他的外在表现完全相反。外表越冷漠的时候，心声越滚烫。他的心声是话痨型的——会自言自语、会吐槽、会纠结、会骂自己。这种反差是全书最大的萌点和笑点来源。

#### 病娇属性的合理呈现（新增）

不是无端的占有欲，而是源于"害怕失去"的创伤后应激。表现为：
- 心声中反复出现"不要离开我""如果你走了我会怎样"的焦虑循环
- 在沈雾与其他男性接触时心声失控（但行为上保持克制）
- 在极端情况下（沈雾受重伤）会展露出"我要毁了所有伤害你的人"的阴暗面

#### 暗中势力

| 势力 | 说明 | 首领 |
|------|------|------|
| 影卫 | 母后留下的暗卫组织 | 影七，共十二人 |
| 谢家暗线 | 舅父谢峥（兵部尚书）暗中部署的接应点 | 谢峥 |
| 旧部 | 太子在位时提拔的一批中下层官员，分布各地 | 各地官员 |

### 2.3 关键配角

#### 柳氏（太子侧妃）

- **身份**：萧渊的侧妃，三皇子萧承的眼线
- **核心特质**：她的背叛真相比表面更复杂——她确实是萧承的人，但她对萧渊也曾有过真心。她最终的选择是在"保全家族"和"对萧渊的愧疚"之间挣扎
- **出场**：第1章（背叛），后续通过回忆和传闻出现

#### 沈夫人（女主继母）

- **身份**：礼部尚书继室，女主继母
- **性格**：势利虚荣，善于算计
- **动机**：推动和离，让女主让位给亲生女儿

#### 沈婉（沈家庶妹）

- **身份**：沈雾的庶妹，沈夫人的亲生女儿
- **性格**：表面天真烂漫，实际嫉妒心强
- **出场**：第2章（嘲讽）

#### 青黛（女主陪嫁丫鬟）

- **身份**：女主陪嫁丫鬟，忠心耿耿
- **性格**：活泼机灵，嘴甜心善，忠心但不愚忠，有自己的判断力
- **作用**：女主的情感倾诉对象，提供喜剧调剂
- **副线**（新增）：她与影七的感情线是"冰与火"的碰撞，影七奉命监视沈雾，青黛发现后与影七产生冲突

#### 影七（男主暗卫首领）

- **身份**：男主暗卫组织"影卫"首领
- **性格**：冷面话少，但对萧渊绝对忠诚
- **隐藏任务**（新增）："监视沈雾确认她不是萧承的人"，这个任务在中期被青黛发现，引发副线冲突

#### 谢婉儿（谢家小姐）

- **身份**：谢家小姐，萧渊的表妹
- **性格**：聪明、骄傲、对萧渊有青梅竹马的感情
- **复杂动机**（新增）：她不是恶毒的情敌，而是一个"站在家族利益和个人感情交叉路口"的复杂角色。她被谢家派来黔州，明面上是照顾表兄，实际任务是确保萧渊与谢家的利益绑定——如果萧渊复位，谢家希望皇后是谢家女儿
- **出场提前**：第7章（流放路上）

#### 苗疆圣女·月鸢

- **身份**：苗疆现任圣女，与沈雾生母是同族
- **作用**：她知晓同心蛊的一切，包括副作用和解除方法。她的出场将带来关于沈雾身世的核心真相

#### 萧承（反派）

- **身份**：三皇子，现任监国
- **核心特质**（新增）：不是脸谱化的反派。他的动机有其合理性——赵贵妃之子，从小被母族教育"皇位本该属于你"，他真心认为自己比萧渊更适合做皇帝。他的狠辣是在宫廷权力斗争中被逼出来的
- **弱点**：他对赵贵妃有真实的孝心，这是后期可以被利用的裂缝

## 四、核心情感节奏设计

### 亲密接触递进表

| 章节 | 接触类型 | 触发情境 | 萧渊心声 |
|------|----------|----------|----------|
| 第5章 | 靠肩而眠 | 破庙避雨，沈雾冷到发抖 | "别动……就让我这样待一会儿" |
| 第9章 | 握手暖手 | 萧渊受伤，无意识抓住沈雾的手 | "别走……不要离开我" |
| 第12章 | 被迫同床 | 荒院只有一张床 | "她好香……不行要克制" |
| 第15章 | 擦药/包扎 | 萧渊背部受伤，沈雾帮忙上药 | "她的手指……我要疯了" |
| 第18章 | 额头轻触 | 萧渊为沈雾擦泪 | "想吻她……不是现在" |

### 心声使用原则

1. **不滥用**：不是每个场景都有心声，只在情绪波动时触发
2. **递进式**：前期心声主要是"关心/担忧"，中期出现"占有欲/阴暗面"，后期出现"恐惧失去/脆弱"
3. **信息差**：心声透露的信息要领先于情节发展，让读者知道但角色不知道，制造"快告诉她啊！"的焦急感
4. **反转用**：关键时刻心声内容与读者预期相反，制造震撼

### 戏剧张力增强
