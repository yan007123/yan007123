# 《和离后，我听见了废太子的心声》1-20章简易细纲

## 故事核心脉络

**第一卷·流离（第1-20章）**

| 阶段 | 章节 | 核心主题 | 情感推进 | 权谋推进 |
|------|------|----------|----------|----------|
| **和离与追随** | 1-3章 | 从被抛弃到主动追随 | 恐惧被抛弃→发现被爱→决定追随 | 萧渊被废，暗中布局 |
| **流放路上** | 4-10章 | 追杀与磨合 | 陌生→心动→信任建立 | 萧承追杀，萧渊联络旧部 |
| **扎根黔州** | 11-20章 | 站稳脚跟 | 信任→感情升温→危机初现 | 萧渊积蓄力量，准备反击 |

## 第一章：和离

| 项目 | 内容 |
|------|------|
| **核心情节** | 萧渊被废，与沈雾和离。玉坠发烫，沈雾意外激活同心蛊，听到萧渊内心真实想法——他并非无情，而是为了保护她。 |
| **关键场景** | 太子府正厅，沈雾跪地签和离书，玉坠发烫触发心声 |
| **心声内容** | "不要签""求你，不要签""和离是唯一的办法""对不起，这三年是我负了你" |
| **沈雾反应** | 震惊、困惑、眼眶泛红但未落泪，意识到他冷漠外表下的真实想法 |
| **章末钩子** | 萧渊心声："下辈子，我不会再让你嫁给我。" |
| **信息差** | 沈雾知道萧渊的真实想法，但萧渊以为她恨他 |

## 第二章：无处可归

| 项目 | 内容 |
|------|------|
| **核心情节** | 沈雾回到沈家，遭遇继母方氏和庶妹沈婉的冷嘲热讽。方氏逼迫她改嫁，沈父因"克父克夫"的偏见急于摆脱她。沈雾通过心声确认萧渊即将被流放黔州。 |
| **关键场景** | 沈家正厅家宴，方氏提出三个改嫁人选；沈父暗示"克夫"命格；沈婉嘲讽"三年不碰你" |
| **家族冲突** | 方氏表面关切实则嫌弃；沈父因"克父克夫"偏见视她为灾星，急于再嫁；沈婉幸灾乐祸 |
| **心声内容** | "明日启程……黔州……三年……沈雾保重……不会再见了" |
| **沈雾反应** | 冷静应对，拒绝改嫁，暗中决定追随萧渊 |
| **章末钩子** | 沈婉透露：废太子明日流放黔州，"十个有九个回不来" |
| **伏笔埋设** | 母亲遗物绣帕（林家信物，苗疆花纹）；《滇南济世指迷》医书；信中"苗疆……林家……绣帕……" |

## 第三章：千里追随

| 项目 | 内容 |
|------|------|
| **核心情节** | 沈雾带着青黛追上流放队伍，在雪地里与萧渊重逢。萧渊表面冷漠让她回去，心声暴露惊喜和担忧。沈雾坚持跟随，萧渊无奈默许。 |
| **关键场景** | 官道雪地，沈雾拦住囚车；萧渊冷漠拒绝但心声狂喜 |
| **心声内容** | "她来了？她怎么来了？沈家那些人……她是不是被赶出来的？该死……她跟着我吃苦……" |
| **沈雾反应** | 坚定表示"我无处可去"，（前期展示医术，提及自幼随母学医经历） |
| **押送官** | 赵瑾瑜（赵明远之子）怀疑沈雾，暗中监视 |
| **章末钩子** | 萧渊心声："她来了……也好……至少我能看着她……" |
| **信息差** | 萧渊以为沈雾是被沈家赶出来，不知她是听到心声后主动追随 |

## 附录：核心设定回顾

### 同心蛊规则

| 规则 | 说明 |
|------|------|
| 单向传递 | 沈雾能听到萧渊心声，萧渊无法感知 |
| 自动触发 | 沈雾无法控制开关，萧渊情绪波动时自动触发 |
| 真实传递 | 蛊虫只传递"真实想法"，无法被伪装或欺骗 |
| 距离限制 | 10米内最强，超过1000米无效 |
| 副作用 | 长期接收会导致精神损耗，最终失去自我意识 |

### 心声使用原则

1. **不滥用**：不是每个场景都有心声，只在情绪波动时触发
2. **递进式**：前期"关心/担忧"，中期"占有欲/阴暗面"，后期"恐惧失去/脆弱"
3. **信息差**：心声透露的信息领先于情节发展，制造"快告诉她啊！"的焦急感
4. **反转用**：关键时刻心声内容与读者预期相反，制造震撼
