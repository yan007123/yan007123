## 第一章：和离

| 项目 | 内容 |
|------|------|
| **时间** | 乾元十四年，腊月初九 |
| **地点** | 太子府正厅 |
| **核心情节** | 萧渊被废为庶人，即将流放黔州。在太子府正厅与沈雾和离。柳氏（侧妃）先离开，手里也拿着和离文书——说明柳氏同样被和离（为摆脱赵贵妃眼线）。沈雾站了一炷香时间后进入，萧渊让她签和离书。玉坠发烫，沈雾意外激活同心蛊，听到萧渊内心真实想法——他并非无情，而是为了保护她。 |
| **关键场景1** | 柳氏身形颤抖离开，拿文书——暗示柳氏也被和离 |
| **关键场景2** | 萧渊坐在上首，茶凉透了，不看沈雾，只说"签了""我没有耐心等你" |
| **关键场景3** | 萧渊外貌细节：身穿洗得发白的玄色常服，领口松了两颗扣子，露出苍白锁骨，锁骨下方有旧疤（几年前为护先帝留下的）。他像一把在鞘中锈了三年、等着出鞘时刻的剑 |
| **关键场景4** | 沈雾跪地签和离书时，玉坠突然滚烫，触发心声 |
| **心声内容（完整版）** | 【不要签】【求你，不要签】→【孤守孝三年，这三年让她受委屈了。那柳氏却一直是赵贵妃和萧承一党的眼线，这次和离正好摆脱这个眼线。】→【和离……是目前唯一的办法。她回到沈家，沈家应该会护着她。如今沈家已经向赵皇后和萧承一党示好，只要她不再是太子妃...应该不会为难她。】→【……对不起。】【沈雾，对不起。这三年来因为柳氏也一直怀疑你。】→【别哭。】【你千万别哭...】（此时沈雾眼眶泛红）→沈雾签字后：【……好】【我被废为庶人，马上要被流放黔州，这样最好。】【她会没事的。】【她会没事的……】（反复）→关于玉坠：【她会不会把玉坠还给我？】【不，让她留着。那是母后的遗物，但我给了她，就是她的了。】【就当……唯一的念想吧。】→最后：【走远一点……就安全了……】【沈雾……】【下辈子，我不会再让你嫁给我。】 |
| **沈雾反应** | 握笔手僵、瞳孔骤缩、手指发抖、眼眶泛红但强忍不哭、稳住手签字、按胸口玉坠、不回头离开太子府 |
| **萧渊行为** | 取过和离书扫一眼折好放袖中（始终未看她一眼）、唤青黛"送你家姑娘回沈府"、背对沈雾大步走出正厅 |
| **章末钩子** | 沈雾站在空荡正厅，按胸口玉坠（温度退去），最后一声心声消失：【下辈子，我不会再让你嫁给我。】沈雾闭眼再睁眼，眼底清明——她不知道那声音是什么，但知道那个冷漠递上和离书的男人，心里说的是"不要走"。 |
| **信息差** | 沈雾知道萧渊的真实想法（通过心声），但萧渊以为她恨他；萧渊不知道沈雾能听到他的心声 |
| **伏笔埋设** | 玉坠是谢皇后遗物，新婚之夜给沈雾；同心蛊激活条件：萧渊情绪剧烈波动（极度痛苦）；柳氏是赵贵妃和萧承的眼线 |

## 第二章：无处可归

| 项目 | 内容 |
|------|------|
| **时间** | 腊月初九（回沈家当晚）+ 腊月初十（次日） |
| **地点** | 沈家（大门、前院、正厅、后院厢房） |
| **核心情节** | 沈雾回到沈家，遭遇继母方氏和庶妹沈婉的冷嘲热讽。方氏逼迫改嫁，沈父因"克父克夫"的偏见急于摆脱她。沈雾通过心声确认萧渊即将流放黔州。最终决定追随萧渊。 |
| **关键场景1** | 沈家大门比太子府矮一半，门房刘叔犹豫后才开门；丫鬟们看到沈雾都低头走开，窃窃私语 |
| **关键场景2** | 方氏穿石榴红褙子、簪赤金步摇，提前梳妆在正厅等着（表面关心实际嫌弃）；拉手力道太轻像嫌脏 |
| **关键场景3** | 当天晚上方氏摆家宴"接风"，沈桓出席（面前酒杯始终没端起来） |
| **关键场景4** | 方氏提出三个改嫁人选（具体）：①江南道知府刘大人（47岁，无嫡子，嫁过去是当家主母）②苏州丝绸商贾李家（商户但万贯家财）③娘家侄子方瑞（20出头纨绔子弟，前年堵过丫鬟）——方氏最想嫁的是方瑞 |
| **关键场景5** | 沈桓发怒："你已被废太子休弃"→沈雾纠正："是和离，不是休弃"；沈桓说出真相："你出生那年我就被人弹劾，险些丢了官职……'克父克夫'的命格，你还要拖累沈家到几时？" |
| **关键场景6** | 沈桓让沈雾回去，头也不回走了；方氏私下威胁："你是聪明人。你在沈家没有靠山了……趁你还有几分姿色，赶紧找个人家嫁了。别拖累沈家。" |
| **关键场景7** | 住处安排：后院二层小楼最偏的一间厢房（原放杂物的，方氏特意安排） |
| **关键场景8** | 次日一早沈婉来访（鹅黄色袄裙、抱暖炉），嘲讽："三年都不碰你，那你跟守活寡有什么区别呀？"；透露："废太子明天就要被押送去黔州了，流放三千里呢……十个有九个回不来。" |
| **关键场景9** | 沈雾发现母亲遗物：绣帕（苗疆林家特有花纹——藤蔓缠绕展翅之蝶）+ 信（模糊字迹："……苗疆……林家……绣帕……"） |
| **关键场景10** | 药箱内容详细列举：银针、几包常备草药、《滇南济世指迷》（外祖父林远山所赠）、《大晟惠民局方·初稿》（太医局女医官淳于衍赠送） |
| **关键场景11** | 重要物品：床头上不知谁放的青布包碎银子 |
| **心声内容（完整版）** | （第二章末尾玉坠微热）【……明日……启程……黔州……】【……三年……】【……沈雾……保重…这三年是我负你了…不会再见了……】（微弱如风中残烛）最后一句清晰：【……好好活着。】中间还有一段：【这沈家的院墙也不高……不，不能。】【她应该很恨我……对不起……】（暗示萧渊曾在院外徘徊） |
| **沈雾反应** | 表面冷静应对（屈膝行礼、"多谢母亲费心"、"暂时没有再嫁打算"），内心坚定；将玉坠取下查看（除温热外无异常）；赤足到窗边推开缝看雪地；让青黛次日打听出发情况 |
| **章末钩子** | 院墙外雪地里一行脚印延伸到黑暗深处；沈雾决定追随——"青黛，你明日早点去打听一下他出发的情况" |
| **伏笔埋设** | 苗疆林家（外祖家）、绣帕信物、《滇南济世指迷》医书、黔州离苗疆不远（为后期寻亲解蛊铺垫）、碎银子来源未知 |

## 第三章：千里追随（完整版）

| 项目 | 内容 |
|------|------|
| **时间** | 乾元十四年，腊月十一（清晨出发→巳时追上队伍） |
| **地点** | 沈家→永定门外→囚车车厢 |
| **核心情节** | 沈雾决定追随萧渊去黔州。青黛打听到流放队伍出发信息。沈雾整理包袱准备出发。青黛从质疑到支持。**追赶队伍→雪地重逢→上车互动** |
| **关键场景1** | 青黛打听到的消息（详细）：萧渊车驾辰时出城、巳时出发、走南门（永定门）、往南去；押送官赵虎；手下二十来个兵丁 |
| **关键场景2** | 沈雾整理包袱内容（详细）：几件换洗衣裳、母亲留下的绣帕和那封信、简易药箱（银针、几包常备草药、《滇南济世指迷》、《大晟惠民局方·初稿》）、床头的碎银子（青布包着） |
| **关键场景3** | 青黛质问："小姐……你不会是要去黔州吧？黔州啊！三千里！瘴气毒虫！就算殿下对你有心……可他对你有心吗？" |
| **关键场景4** | 沈雾回答（关键台词）："我还有其他去处吗？"（停顿）"黔州离我外祖家很近。"（轻声但确定）（内心独白：可能他有心呢？我还需要确认一下。） |
| **关键场景5** | **追赶队伍**：沈雾和青黛赶往永定门，在雪地里找到正在修车轮的囚车 |
| **关键场景6** | **雪地重逢**：沈雾上前对赵虎说："民女沈雾，是萧渊故人。我二人想随行前往黔州。"赵虎质疑："什么故人？流放之人还有故人？" |
| **关键场景7** | **萧渊开口**："她是我的人。让她们跟着。"（表面冷漠）内心：【你怎么来了？！你疯了吗？黔州三千里！】 |
| **关键场景8** | **上车互动**：沈雾上车与萧渊面对面坐着（膝盖距离不到一尺）。萧渊表面闭眼，内心疯狂吐槽：【她坐得离我太近了】【该死的，我没洗澡】【萧渊，你真是个废物】 |
| **关键场景9** | **心声反差**：萧渊关心沈雾冷暖，想给她衣服但只有囚衣。提到影七已提前去驿站安排。沈雾听到心声强忍笑意 |
| **关键场景10** | **沈雾内心决定**：三千里，两个月，有的是时间弄清楚玉坠秘密，弄清楚萧渊嘴上说的和心里想的为什么判若两人 |
| **本章结束点** | 马车辘辘向前，碾过一地残雪。沈雾的心前所未有地安定 |
| **与原大纲差异说明** | **无差异** - 本章完整包含了追赶、重逢、上车互动的完整情节，与原大纲第三章内容一致 |

## 附录：核心设定回顾

### 同心蛊规则

| 规则 | 说明 |
|------|------|
| 单向传递 | 沈雾能听到萧渊心声，萧渊无法感知 |
| 自动触发 | 沈雾无法控制开关，萧渊情绪波动时自动触发 |
| 真实传递 | 蛊虫只传递"真实想法"，无法被伪装或欺骗 |
| 距离限制 | 10米内最强，超过1000米无效 |
| 副作用 | 长期接收会导致精神损耗，最终失去自我意识 |

### 心声使用原则

1. **不滥用**：不是每个场景都有心声，只在情绪波动时触发
2. **递进式**：前期"关心/担忧"，中期"占有欲/阴暗面"，后期"恐惧失去/脆弱"
3. **信息差**：心声透露的信息领先于情节发展，制造"快告诉她啊！"的焦急感
4. **反转用**：关键时刻心声内容与读者预期相反，制造震撼
